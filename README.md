# Rejection Game

This is my implementation of the rejection app for DevAnywhere.io and
ericelliottjs.com.

## Commands

Run the app locally:

```bash
yarn start
```

Run unit tests:

```
yarn test
```

## Showcase

This app shows:

- Unit tests with RITEway
- Styling with Material-UI
- Redux
- React Hooks
